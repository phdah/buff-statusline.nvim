## Requirements

- Neovim 0.X

## Installation

Use your favorite package manager, e.g., `packer`:
````lua
use {'phdah/buff-statusline.nvim'}
````
and in your `init.lua``, put
````lua
require('buff-statusline.nvim').setup()
````
